# Chrome/Edge 스토어 제출 체크리스트

## 1. 패키지 상태
- Manifest V3 사용
- 원격 코드 사용 없음
- 권한 최소화: `clipboardRead`, `clipboardWrite`만 사용
- 아이콘 파일 준비: 16/32/48/128
- 기본 동작: 공백 기반 텍스트를 탭 구분으로 변환해 복사
- 옵션 동작:
  입력 구분자(탭, 세미콜론, 공백, 콤마, 기타 1자)
  출력 구분자(탭, 세미콜론, 공백, 콤마, 제거, 기타 1자)
- 안전 동작:
  입력/출력 구분자가 같으면 변환 버튼 비활성화
  `기타` 선택 상태에서 값이 비어 있으면 변환 버튼 비활성화
- 결과 미리보기 동작:
  출력 구분자가 `탭`일 때만 다중 셀 표시
  그 외 출력 구분자(세미콜론/공백/콤마/제거/기타)는 단일 셀 표시
- 연속 작업 동작:
  변환 성공 후에도 상단 버튼/옵션 UI를 유지해 연속 실행 가능
- 옵션 저장: 사용자가 고른 입력/출력 구분자 및 기타 1자 값을 기기 로컬에 저장 후 다음 실행 시 복원

## 2. 제출 전 파일 압축
아래 런타임 파일/폴더만 포함해서 zip을 만드세요.
- `manifest.json`
- `popup.html`
- `popup.js`
- `icons/`
- `fonts/`

권장(스토어 심사 참고용):
- `PRIVACY_POLICY.md`

PowerShell 예시:

```powershell
if (Test-Path "dist/delimiter-copy-converter-v1.0.1.zip") {
  Remove-Item "dist/delimiter-copy-converter-v1.0.1.zip"
}

Compress-Archive -Path `
  manifest.json, popup.html, popup.js, icons, fonts `
  -DestinationPath dist/delimiter-copy-converter-v1.0.1.zip `
  -CompressionLevel Optimal
```

## 3. 스토어 입력용 문구 (권한 정당화)
### Short description
클립보드 텍스트 구분자를 빠르게 변환해 다시 복사하는 생산성 확장 프로그램입니다.

### Detailed description
- 기본값은 공백→탭 변환으로 엑셀/시트 붙여넣기에 바로 사용할 수 있습니다.
- 옵션 버튼에서 입력/출력 구분자를 직접 선택할 수 있습니다.
- 출력 구분자에 `제거`를 선택하면 입력 구분자를 삭제하는 방식으로 변환할 수 있습니다.
- `기타` 구분자는 1글자 입력으로 지원됩니다.
- 입력/출력 구분자가 같으면 변환 버튼이 비활성화되어 불필요한 실행을 막습니다.
- 마지막으로 선택한 옵션(입력/출력 구분자, 기타 1자)은 로컬에 저장되어 다음 실행에도 유지됩니다.
- 모든 처리는 로컬 브라우저에서만 수행됩니다.

### Single purpose
사용자가 클립보드 텍스트의 구분자를 원하는 형식으로 즉시 변환해 다시 복사할 수 있도록 돕는 생산성 도구입니다.

### Permissions justification
- `clipboardRead`: 사용자가 버튼을 눌렀을 때 현재 클립보드 텍스트를 읽어 변환하기 위해 필요합니다.
- `clipboardWrite`: 변환된 텍스트를 클립보드에 다시 복사하기 위해 필요합니다.

### Remote code use
No. 확장 프로그램은 원격으로 호스팅된 코드를 실행하지 않습니다.

### Data usage
- 사용자 데이터 수집: 없음
- 데이터 판매/공유: 없음
- 데이터 외부 전송: 없음
- 클립보드 텍스트는 로컬 브라우저 내에서만 처리
- 옵션 설정값(입력/출력 구분자, 기타 1자)은 기기 로컬 스토리지에만 저장

## 4. 수동 테스트 체크리스트 (제출 전)
- 기본 상태에서 공백→탭 변환 정상 동작
- 옵션 열기/닫기 토글 정상 동작
- 입력/출력 구분자 동일 선택 시 변환 버튼 비활성화
- `기타` 선택 시 입력칸 노출 및 1글자만 입력되는지 확인
- `기타`를 비워두면 변환 버튼 비활성화되는지 확인
- 출력 구분자 `제거` 선택 시 입력 구분자가 실제로 제거되는지 확인
- 결과 미리보기가 `탭` 출력일 때만 다중 셀, 그 외에는 단일 셀인지 확인
- 변환 성공 후에도 상단 `옵션`/`변환 후 복사` 버튼이 유지되는지 확인
- 옵션 값을 변경한 뒤 팝업을 다시 열어도 동일 값으로 복원되는지 확인

## 5. 이미지 준비 (Chrome Web Store / Edge Add-ons 공통 권장)
- 스크린샷 1장 이상 (권장 1280x800)
- 프로모션 이미지 1장

스토어에는 URL 입력이 필요할 수 있으므로 아래 중 하나에 `PRIVACY_POLICY.md` 내용을 게시해 URL을 준비하세요.
- GitHub Pages
- Notion 공개 페이지
- 개인 웹사이트

## 7. Edge 배포 참고
- Edge(Chromium)는 현재 manifest/코드 그대로 호환됩니다.
- Chrome에 등록 후 Edge Add-ons에서 동일 zip으로 제출 가능합니다.
- 스토어 설명 문구만 Edge용으로 약간 조정하면 됩니다.
