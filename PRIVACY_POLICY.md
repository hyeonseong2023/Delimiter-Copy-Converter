# 개인정보처리방침 (공백→탭 변환기)

최종 업데이트: 2026-03-20

## 1. 개요
공백→탭 변환기(이하 "확장 프로그램")는 사용자가 버튼을 누를 때 클립보드 텍스트를 읽어 선택한 구분자로 변환한 뒤 다시 클립보드에 복사하는 기능을 제공합니다.
기본 변환은 공백→탭이며, 옵션에서 입력/출력 구분자를 선택할 수 있습니다.

## 2. 수집하는 정보
확장 프로그램은 서버로 전송하거나 영구 저장하기 위한 목적으로 개인정보를 수집하지 않습니다.

클립보드 권한(`clipboardRead`, `clipboardWrite`)은 아래 동작을 위해서만 사용됩니다.
- 사용자가 버튼 클릭 시 클립보드 텍스트 읽기
- 변환된 텍스트를 클립보드에 다시 쓰기
- 사용자가 선택한 구분자 값(옵션/기타 1자)으로 로컬에서만 변환 처리

## 3. 데이터 처리 방식
- 클립보드 데이터는 사용자의 로컬 브라우저 내에서만 처리됩니다.
- 외부 서버 전송, 제3자 공유, 광고/분석 도구 연동을 하지 않습니다.
- 확장 프로그램은 별도의 사용자 계정, 로그인, 추적 기능을 제공하지 않습니다.

## 4. 데이터 보관
- 클립보드 데이터는 변환 처리 직후 메모리에서만 사용되며 별도 저장소(로컬/원격 DB)에 보관하지 않습니다.
- 사용자가 선택한 옵션 값(입력/출력 구분자, 기타 1자)은 사용성 편의를 위해 브라우저 로컬 스토리지에 저장됩니다.
- 해당 옵션 저장 데이터는 서버로 전송되지 않으며, 사용자가 확장 프로그램을 제거하거나 브라우저 저장소를 정리하면 삭제됩니다.

## 5. 제3자 제공
- 본 확장 프로그램은 어떠한 사용자 데이터도 제3자에게 제공하지 않습니다.
