## 2) Chrome Web Store 붙여넣기
### Short description
클립보드 텍스트 구분자를 원하는 형식으로 빠르게 변환해 다시 복사하는 생산성 확장 프로그램입니다.

### Detailed description
클립보드 텍스트의 구분자를 바로 변환해 다시 복사합니다.

- 기본값은 공백 → 탭 변환입니다.
- 옵션에서 입력/출력 구분자를 선택할 수 있습니다.
  입력: 탭, 세미콜론, 공백, 콤마, 기타 1자
  출력: 탭, 세미콜론, 공백, 콤마, 제거, 기타 1자
- 출력을 `제거`로 선택하면 입력 구분자를 삭제하는 방식으로 변환됩니다.
- `기타`는 1글자만 입력됩니다.
- 입력/출력 구분자가 같으면 변환 버튼이 비활성화됩니다.
- 마지막으로 선택한 옵션값(입력/출력 구분자, 기타 1자)은 기기 로컬에 저장되어 다음 실행 시 복원됩니다.
- 결과 미리보기는 출력이 `탭`일 때만 여러 셀로 표시되며, 그 외 출력은 단일 셀로 표시됩니다.
- 모든 변환은 로컬 브라우저에서만 처리되며 외부 서버로 전송하지 않습니다.

### Single purpose
사용자가 클립보드 텍스트의 구분자를 원하는 형식으로 변환해 다시 복사할 수 있도록 돕는 생산성 도구입니다.

### Permissions justification
- `clipboardRead`: 사용자가 버튼을 눌렀을 때 클립보드 텍스트를 읽어 구분자를 변환하기 위해 필요합니다.
- `clipboardWrite`: 변환된 텍스트를 클립보드에 다시 복사하기 위해 필요합니다.

### Privacy practices / Data usage 설명용 문구
- 클립보드 텍스트는 사용자 기기에서만 변환 처리되며 서버로 전송되지 않습니다.
- 옵션값(입력/출력 구분자, 기타 1자)은 사용자 편의를 위해 브라우저 로컬 스토리지에만 저장됩니다.
- 사용자 데이터를 판매, 공유, 광고 목적으로 사용하지 않습니다.

## 3) Edge Add-ons 붙여넣기
### Description
클립보드 텍스트 구분자를 변환해 다시 복사하는 생산성 확장 프로그램입니다.
기본값은 공백 → 탭이며, 옵션에서 입력/출력 구분자를 선택할 수 있습니다.
출력 구분자 `제거`를 통해 입력 구분자를 삭제하는 변환도 지원합니다.
입력/출력 구분자가 같으면 변환 버튼이 비활성화됩니다.
모든 처리는 로컬 브라우저에서만 수행됩니다.

### Notes for certification
아래 순서로 동작 확인 부탁드립니다.

1. 확장 프로그램 팝업을 열면 상단에 `옵션`, `변환 후 복사` 버튼이 표시됩니다.
2. `옵션` 클릭 시 입력/출력 구분자 선택 UI가 열립니다.
3. 기본값은 입력 `공백`, 출력 `탭`입니다.
4. 클립보드에 `A B C`를 복사한 후 `변환 후 복사` 클릭 시, 클립보드 값이 `A\tB\tC`로 변환됩니다.
5. 변환 성공 후에도 상단 버튼(`옵션`, `변환 후 복사`)은 유지되어 연속 작업이 가능합니다.
6. 입력/출력 구분자를 동일하게 선택하면 `변환 후 복사` 버튼이 비활성화됩니다.
7. `기타` 선택 시 입력칸이 표시되고 1글자만 입력 가능합니다.
8. 출력 구분자 `제거` 선택 시 입력 구분자가 삭제되는 변환이 동작합니다.
9. 결과 미리보기는 출력이 `탭`일 때만 다중 셀, 그 외에는 단일 셀로 표시됩니다.
10. 옵션값 변경 후 팝업을 다시 열면 이전 선택값이 복원됩니다.
